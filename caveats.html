Estimate Caveats

  • Database considerations are not included in this estimate.
  • Estimates are based on server moves only, lift-and-shift migrations. If multiple apps are on each server, complexity/cost will increase.
  • Assumes operating system will remain the same through the move, OS upgrades or OS changes will add to cost.
  • Assumes that packages like PHP and other software will remain the same though the move
  • Additional hours for Go-Live support is not included.
  • Estimates do not include any licensing costs for OSs, Databases, or migration tools
